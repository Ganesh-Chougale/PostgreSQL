# Data Definition Language

## . Create Table
> Syntax
```bash
  CREATE TABLE table_name (
    column_name data_type [constraints]
    );
```
> Example
```sql
CREATE TABLE employees (
    employee_id INT PRIMARY KEY,
    first_name VARCHAR(50) NOT NULL,
    last_name VARCHAR(50) NOT NULL,
    hire_date DATE
);
```
